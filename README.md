<h2>Catalog App - SwiftUI - MVVM</h2>
An catalog app to browse catalogs of various brands following MVVM pattern.
<h2>UI Frameworks</h2>
<ul>
 	<li>SwiftUI</li>
 	<li>MapKit</li>
</ul>
<h2>Backend, Networking and Services</h2>
<ul>
 	<li>URLSession</li>
 	<li>Corelocation</li>
</ul>
